Instructions

This extension is invoked when the text in the omnibar is prefixed with !f. Everything beyond it is sent to the extension. The first word is used to identify the resolver which is mapped to a URL which can directly trigger a search. For example,

!f cr tokio

Will be parsed as the cr resolver and the search string as tokio and omnibox will be replaced by
https://crates.io/crates/tokio.
